Natural Breaks

This cholorpleth divides the data into naturally occurring sections. A pro is that this classification identifies natural classes within the data, creating choropleths with accurate data trends. A con is that it is not well suited for data with low variance.

Equal Count (quantile)

This cholorpleth divides the data into 6 different classes, with the same number of municipalities per class/ color. Since Puerto Rico has 78 municipalities, there are 13 municipalities per color. A pro is that each class is equally represented on the choropleth. A con is that it may group outliers in with other classes, making it a poor representation of the actual data distribution.

Pretty Breaks

This cholorpleth divides the data into classes with round numbers ending in zeros. A pro is that it represents data in an easy to understand manner, since each of the class boundaries are split into equal sizes that end in whole numbers. A con is that some classes may be underrepresented and outliers may be grouped into classes that don't accurately represent data trends.
